## 주요 코드

### 1. IAPV5Manager.cs

Unity IAP V5 기반 인앱결제 처리 코드입니다.

- 상품 등록 및 구매 가능 상태 확인
- 구매 요청 전 App Store / Google Play 상품 상태 체크
- 구매 성공 / 실패 / 보류 상태 처리
- 구매 완료 후 보상 지급
- 이미 보유한 상품, 미지급 보상 등 예외 상황 대응
- 구매 확정 처리

실제 라이브 서비스 과정에서 인앱결제 오류, 보상 미지급, 자동 환불 가능성 등을 겪으며 안정성을 높이기 위해 개선한 코드입니다.

### 2. IapRewardExample.cs

인앱결제 성공 이후 실제 게임 보상을 지급하는 예시 코드입니다.

- 상품 ID 기반 보상 분기
- 다이아, 패키지, 광고 제거 등 상품 유형별 처리
- 중복 보상 방지
- 구매 후 저장 처리
- UI 갱신 흐름 예시

전체 프로젝트 코드를 공개하지 않고도 인앱결제 보상 구조를 이해할 수 있도록 예시 형태로 정리했습니다.
