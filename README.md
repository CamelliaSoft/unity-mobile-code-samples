# Unity Mobile Code Samples

Unity 기반 모바일 키우기 게임을 개발·출시·운영하면서 사용한 코드 샘플입니다.

전체 상용 프로젝트를 공개하는 대신, 코드 퀄리티 확인을 위해 인앱결제 처리, 오브젝트 풀링, 라이브 서비스 안정화와 관련된 일부 코드만 정리했습니다.

## 주요 코드

### 1. IAPV5Manager.cs

Unity IAP V5 기반 인앱결제 처리 코드입니다.

- 상품 등록 및 구매 가능 상태 확인
- 구매 요청 전 IAP 초기화 상태 체크
- 구매 실패 상황 처리
- 구매 완료 후 보상 지급
- 중복 보상 지급 방지
- 보상 지급 후 구매 확정 처리

실제 라이브 서비스 과정에서 인앱결제 오류, 보상 미지급, 자동 환불 가능성 등을 겪으며 안정성을 높이기 위해 개선한 코드입니다.

### 2. TouchEffect Object Pooling

터치 이펙트처럼 짧은 시간에 반복 생성되는 UI 효과를 매번 생성/삭제하지 않고 재사용하기 위한 오브젝트 풀링 예시입니다.

- 시작 시 이펙트 오브젝트를 미리 생성
- 터치 시 비활성화된 오브젝트 재사용
- 이펙트 재생 후 FadeOut 처리
- 재생 완료 후 SetActive(false)로 풀에 반환

작은 기능이지만 모바일 환경에서 불필요한 Instantiate/Destroy를 줄이고, 반복 입력 상황에서도 안정적으로 동작하도록 구성했습니다.
